# ViroTikly

ViroTikly is a simple web tool that helps creators upload, manage, and prepare TikTok videos easily.

## Features
- Upload video content
- Paste video links (Google Drive, etc.)
- Connect to TikTok account
- Clean and simple interface

## How to use
1. Open the website
2. Upload your video or paste a video link
3. Add a caption
4. Submit to process your content

## Note
This project is currently being developed and is used for learning and testing TikTok API integration.
